## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/movie_app_react_query.git
```

Go to the project directory

```bash
  cd movie_app_react_query
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
